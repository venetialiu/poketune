# PokeTune

Find your Pokémon based on your Spotify listening habits.

## Link to Live Website
[poketune-five.vercel.app]

## Features
- Spotify login
- Fetches your top tracks + audio features
- Builds your “music profile”
- Matches you to the closest Pokémon using stat similarity
- Shows sprite, type, and short explanation

## Tech Stack

- Frontend: React, Vite
- Backend: Node.js + Express
- APIs: Spotify Web API, PokéAPI
- Language: JavaScript
